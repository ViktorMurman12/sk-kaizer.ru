### 8. Важные SEO-принципы для каждой страницы
[] У каждой страницы должен быть свой уникальный `title`
[] У каждой страницы должен быть свой уникальный `description`
[] Один `h1` на страницу
[] Текст не дублировать между страницами
[] Добавлять FAQ и schema.org там, где это уместно
[] Делать внутреннюю перелинковку между услугами, кейсами и статьями
[] Для каждой услуги добавить фото, этапы, цену или факторы цены
